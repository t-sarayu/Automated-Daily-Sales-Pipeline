# Automated-Daily-Sales-Pipeline
Project Title: Automated Daily Sales Pipeline (n8n + Supabase + QuadraticHQ)
Overview
A fully automated workflow that extracts daily sales reports from email, processes them with n8n, loads the data into Supabase (Postgres cloud), and visualizes KPIs like OTIF%, IF%, and OT% in QuadraticHQ.
This project demonstrates workflow automation, database design, and KPI analytics for real-time operational insights.
Email Inbox → n8n → Supabase → QuadraticHQ → KPI Dashboard
